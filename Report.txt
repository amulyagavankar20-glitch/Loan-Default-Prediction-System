4. In Train data after imputing the null values and checking we see the data is highly imbalanced so we apply SMOTE
5. SMOTE is Synthetic Minority Over-sampling Technique which generates new, synthetic data points for the minority class rather than just copying existing ones.
6. The Step-by-Step Process:
    Select a Point: SMOTE picks a random point from the minority class (let's call it Point A).
    Find Neighbors: It finds the k-nearest neighbors (usually k=5) for Point A.
    Draw a Line: It draws an imaginary line between Point A and one of its neighbors (Point B).
    Place a New Point: It picks a random spot along that line. This spot becomes a new, synthetic data point.
